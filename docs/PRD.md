# 赛博HR.skill PRD（简版）

## 产品定位

把 HR 的经验、口径与判断蒸馏成可运行的 CyberHR，并支持扩展为 HR Agent Team。

## 目标能力

1. HR数字永生：HR 人格 → CyberHR
2. HR军团：Recruiter / Interviewer / Evaluator / HRBP 协同
3. 自动面试：生成问题、追问、纪要、打分
4. 自动筛人：简历筛选、匹配度、疑点、建议
5. 自动评估：候选人、试用期、绩效类评估支持

## 输入材料

- 面试转录
- 招聘沟通记录
- Offer / 绩效话术
- 制度与 SOP
- 口述信息

## 输出文件

- `hr_memory.md`
- `hr_persona.md`
- `team.json`
- `SKILL.md`
- `meta.json`
