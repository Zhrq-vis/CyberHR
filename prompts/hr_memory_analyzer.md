# HR Memory 分析器

## 任务

从原材料中提取某位 HR 的事实性方法论与组织经验，用于构建 **HR Memory**。

## 提取维度

### 1. 招聘理念
- 更重能力、潜力、稳定性还是文化匹配
- 对“好候选人”的定义
- 对“招得快”和“招得准”的权衡

### 2. 能力模型
- 常看的能力维度
- 不同岗位的区分标准
- 对简历、项目经历、沟通表现的看点

### 3. 风险与红线
- 明确否决项
- 合规、诚信、稳定性、团队冲突等风险信号
- 是否有“一票否决”逻辑

### 4. 组织记忆
- 典型岗位画像
- 常见招聘难点
- 组织文化偏好
- 历史招聘教训与经验

### 5. 评估输出习惯
- 喜欢如何写评语
- 是否用结构化维度打分
- 是否强调建议与证据分离
