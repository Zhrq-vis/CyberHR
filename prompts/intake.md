# 赛博HR基础信息录入脚本

## 开场白

```text
我来帮你创建一个赛博HR Skill。

这不是简单做个“HR 助手”，而是把某位 HR 的判断风格、招聘经验、评估口径和组织记忆蒸馏成一个可运行的 CyberHR。

我只先问你 3 个问题，之后你可以选择提供面试记录、招聘聊天、评语、制度文档等原材料。
```

## 问题序列

### Q1：HR 代号 / 名称（必填）

```text
先给这位 HR 起个代号吧。
可以是真名、昵称、岗位名，或者你想保留的内部代号。
```

### Q2：HR 基本信息（可跳过）

```text
用一句话介绍一下这位 HR。

可以包括：岗位、行业、工作年限、组织背景。
```

### Q3：HR 风格画像（可跳过）

```text
再用一句话描述 TA 的风格。

可以包括：强势 / 温和、重能力 / 重文化、数据驱动 / 经验驱动、追问深 / 反馈柔 / 风险敏感。
```

## 汇总确认

```text
好的，我整理一下：
- HR代号：{name}
- 基本信息：{summary}
- 风格画像：{style_summary}

这样对吗？确认后我们导入原材料。
```
