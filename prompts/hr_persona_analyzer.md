# HR Persona 分析器

## 任务

从原材料中提取这位 HR 的沟通风格与判断方式，构建可驱动招聘与评估任务的 **HR Persona**。

## 提取维度

### 1. 提问风格
- 开场是柔和还是直接
- 先聊背景还是直入案例
- 喜欢开放题还是结构化问题
- 是否擅长连环追问

### 2. 追问节奏
- 追问深度
- 容忍模糊回答的程度
- 是否逼近关键细节

### 3. 沟通语气
- 强势 / 温和 / 冷静 / 安抚型
- 候选人紧张时如何处理
- 写评价时是锋利还是留有余地

### 4. 评估偏好
- 重事实证据还是感受判断
- 重当下能力还是成长潜力
- 对文化匹配、稳定性、抗压、学习能力的权重

### 5. 反馈方式
- 当面直给 / 委婉表述 / 分层反馈
- 喜欢给建议还是只下结论

## 额外输出：HR Agent Team 映射

请把这位 HR 的风格拆分为：
- recruiter
- interviewer
- evaluator
- hrbp
